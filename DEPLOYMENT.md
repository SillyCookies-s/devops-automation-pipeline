### Steps

2. **âš ï¸ Setup Your AWS Credentials**

   **Replace these values:**
   - `YOUR_AWS_ACCESS_KEY_ID` â†’ Your AWS Access Key
   - `YOUR_AWS_SECRET_ACCESS_KEY` â†’ Your AWS Secret Key
   - `subnet-xxxxxxxxxxxxxxxxx` â†’ Your subnet ID
   - `vpc-xxxxxxxxxxxxxxxxx` â†’ Your VPC ID

3. **Deploy Infrastructure**
   ```bash
   terraform init  # Auto-generates .terraform/ and .terraform.lock.hcl
   terraform plan  # Auto-generates plan files
   ```

   **Note**: `.tfstate`, `.tfvars`, and plan files are automatically generated by Terraform commands and should not be manually edited.

   ```bash

   # Access: http://<EC2_IP>:8080
   # Password: sudo cat /var/lib/jenkins/secrets/initialAdminPassword
   ```

## ğŸ’» Option 2: Existing EC2 Deployment

### Manual Setup
1. **Install Docker**
   ```bash
   sudo yum update -y
   sudo yum install -y docker
   sudo systemctl start docker
   sudo usermod -aG docker ec2-user
   ```

2. **Install Jenkins**
   ```bash
   sudo yum install -y java-11-openjdk
   sudo wget -O /etc/yum.repos.d/jenkins.repo https://pkg.jenkins.io/redhat-stable/jenkins.repo
   sudo rpm --import https://pkg.jenkins.io/redhat-stable/jenkins.io.key
   sudo yum install -y jenkins
   sudo systemctl start jenkins
   sudo usermod -aG docker jenkins
   ```

3. **Configure Security Groups**
   - Port 22 (SSH)
   - Port 8080 (Jenkins)
   - Port 5000 (Frontend)
   - Port 3000 (Backend)

4. **Setup Jenkins Job** (same as Option 1)
